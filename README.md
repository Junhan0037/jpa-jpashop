## 개발환경

프로젝트 개발 환경은 다음과 같습니다.

* IDE : IntelliJ IDEA Ultimate
* OS : Windows10
* SpringBoot
* Java8
* Gradle
* JPA

해당 repo는 [자바 ORM 표준 JPA 프로그래밍 기본편 - 김영한](https://www.inflearn.com/course/ORM-JPA-Basic) 해당 강의를 듣고 정리한 REPO 입니다.
